# ERP Agro SQL Analysis

Repositorio de procedimientos SQL orientados a análisis y gestión de operaciones en agroindustria.

## 🚀 Contenido

- Indicadores de producción
- Control de stock en packing
- Trazabilidad de pallets
- Validación y consolidación de datos de pedidos
- Reportes de sanidad
- KPIs operativos

## 💻 Tecnologías
- SQL Server
- T-SQL

## 📊 Enfoque
Optimización de procesos, análisis de datos y soporte a la toma de decisiones en entornos ERP agroindustriales.

## ⚠️ Nota
Los scripts han sido anonimizados y simplificados para fines de portafolio.
